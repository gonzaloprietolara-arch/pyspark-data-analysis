# PySpark Scalable Data Processing

## Overview

This project demonstrates scalable data processing using Apache Spark and the PySpark DataFrame API.

The objective is to perform distributed transformations and analytical queries over a structured dataset, leveraging Spark’s execution engine and SQL capabilities.

---

## Technologies Used

- Apache Spark
- PySpark (DataFrame API)
- Spark SQL
- Python

---

## Implemented Processing Steps

- Loading structured dataset into Spark DataFrame
- Data selection and column transformation
- Filtering records based on business rules
- Aggregations using `groupBy` and `agg`
- Creation of temporary views
- Analytical queries using `spark.sql()`

---

## Key Concepts Demonstrated

- Distributed data processing
- Lazy evaluation model in Spark
- DataFrame transformations
- Aggregation at scale
- SQL queries over distributed datasets
- Scalability and cluster execution principles

---

## Example Operations

- Filtering dataset by specific attributes
- Computing aggregated metrics per category
- Executing SQL-based analytical queries over temporary views
- Transforming and enriching dataset columns
